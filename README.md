# url-shortener

## Run in your machine

### Required

- Node.js >= v16

### Install

#### Set up env

```shell
export INSTANCE_ID="${INSTANCE_ID}"
export DATABASE_URL="${DATABASE_URL}"
export URL_SHORTENER_DOMAIN="${URL_SHORTENER_DOMAIN}"
```

Default Envs

- INSTANCE_ID: 1
- SIGN_DASH_DOMAIN: <http://localhost:3000>

#### Install app

```shell
yarn install
```

#### Run App

```shell
yarn dev
```
